Mis proyectos

Social beer

Social Beer

Este proyecto es una red social SPA responsive, misma que permite a cualquier usuario crear una cuenta de acceso y loguearse con ella; crear, editar, borrar y "likear" publicaciones.

-La lógica del proyecto está implementada completamente en JavaScript (ES6+), HTML, CSS y Firebase.

Burger Queen

Burger Quen

Este proyecto fue solicitado por un pequeño restaurante de hamburguesas, que necesitaba una interfaz en la que pudiera tomar pedidos usando una tablet, y enviarlos a la cocina para que se preparen ordenada y eficientemente.

-La lógica del proyecto está implementada completamente en JavaScript (ES6+), HTML y CSS. Y además se utilizó React y Firebase.

Markdown Links

Markdown Links

Esta librería está realizada para una comunidad de código abierto, la cual usando Node.js, lee y analiza archivos en formato Markdown, para verificar los links que contenga y reportar algunas estadísticas.

-Esta librería está implementada en JavaScript para ser ejecutada con Node.js.
